Web Design and Web development

Many people use the two terms web design and web development interchangeably, but they really do have two different meanings. If you are looking for a new job or someone to build your website, you need to know the difference.

Web Design

Web design is the customer-facing part of the website. A web designer is concerned with how a site looks and how the customers interact with it. Good web designers know how to put together the principles of design to create a site that looks great. They also understand about usability and how to create a site that customers want to navigate around in because it is so easy to do.

Web Development

Web development is the back-end of the website, the programming and interactions on the pages. A web developer focuses on how a site works and how the customers get things done on it.
